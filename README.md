# pores-LigandContactsfinder
Proyecto final

### Instalar con pip 
```
pip install biopython
```

# Para ejecutar el script ingrese los archivos con formatos cif/pdb:
```
python.exe main.py .\****.cif .\****.pdb
```

# Alternativamente ingrese la proteina para buscarla automaticamente
```
python.exe main.py 3FAT
```
